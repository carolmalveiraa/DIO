HTML: Linguagem de marcação de hipertexto

HTML (HyperText Markup Language) é o bloco de construção mais básico da Web. Ele define o significado e a estrutura do conteúdo da Web. Outras tecnologias além de HTML são geralmente usadas para descrever a aparência/apresentação ( CSS ) ou funcionalidade/comportamento
( JavaScript ) de uma página da Web.


"Hipertexto" refere-se a links que conectam páginas da web umas às outras, seja dentro de um único site ou entre sites. Links são um aspecto fundamental da Web. Ao carregar conteúdo na Internet e vinculá-lo a páginas criadas por outras pessoas, você se torna um participante ativo na World Wide Web.

O HTML usa "marcação" para anotar texto, imagens e outros conteúdos para exibição em um navegador da Web. A marcação HTML inclui "elementos" especiais, como:

  • Head
  • Title
  • Body
  • Header
  • Footer
  • Article
  • Section
  • P
  • Div
  • Span
  • Img
  • Aside
  • Audio
  • Canvas
  • Datalist
  • Details
Um elemento HTML é separado de outro texto em um documento por "tags", que consistem no nome do elemento cercado por " < " e " > ". O nome de um elemento dentro de uma tag não diferencia maiúsculas de minúsculas. Ou seja, ele pode ser escrito em maiúsculas, minúsculas ou uma mistura delas. Por exemplo, a title tag pode ser escrita como Title, TITLE, ou de qualquer outra forma. No entanto, a convenção e a prática recomendada é escrever tags em minúsculas.

Pré-requisitos

Depois de aprender HTML, você pode prosseguir para aprender sobre tópicos mais avançados, como:

  • CSS

    • e como usá-lo para estilizar HTML (por exemplo, alterar o tamanho do texto e as fontes usadas, adicionar bordas e sombras projetadas, criar o layout da sua página com várias colunas, adicionar animações e outros efeitos visuais).

  • JavaScript

    • e como usá-lo para adicionar funcionalidade dinâmica a páginas da web (por exemplo, encontrar sua localização e representá-la em um mapa, fazer elementos da interface do usuário aparecerem/desaparecerem quando você alterna um botão, salvar dados dos usuários localmente em seus computadores e muito mais)

Conhecendo as tags novas

Sub

O sub elemento deve ser usado apenas por razões tipográficas, ou seja, para alterar a posição do texto para cumprir convenções ou padrões tipográficos, e não apenas para fins de apresentação ou aparência.

Exemplo:

De acordo com os cálculos de Nakamura, Johnson e Mason 1 este resultará na aniquilação completa de ambas as partículas.

Sup

O sup elemento deve ser usado apenas por motivos tipográficos, ou seja, para alterar a posição do texto para cumprir convenções ou padrões tipográficos, e não apenas para fins de apresentação ou aparência.

Exemplo:

Uma das equações mais comuns em toda a física é E=mc2.

Blockquote

O blockquote Uma URL que designa um documento ou mensagem de origem para as informações citadas. Este atributo tem a intenção de apontar para informações que explicam o contexto ou a referência para a citação.

Exemplo:

A topologia de conexão é limitada a um único caminho ponto a ponto para cada operadora, usado com operadoras padrão, mas muitas operadoras podem ser usadas sem interferência significativa entre si, fora do início da primavera. Isto é por causa do espaço etéreo 3D disponível para os portadores, em contraste com o éter 1D usado pelo IEEE802.3. As portadoras têm uma colisão intrínseca sistema de prevenção, o que aumenta a disponibilidade.

O Del representa um intervalo de texto que foi excluído de um documento. Isso pode ser usado ao renderizar "track changes" ou informações de diff de código-fonte.
